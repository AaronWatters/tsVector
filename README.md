# tsVector
Quick and dirty and minimalist vector and matrix operations in Typescript.  KISS.

This is an experiment in making Typescript based libraries using Vite, initially
ported from

https://github.com/AaronWatters/qd_vector

It is a bit like a "hello world" typescript library implementation.

## Development

### Running the dev server

Install dependencies first if you haven't already:

```bash
npm install
```

Start the Vite development server:

```bash
npm run dev
```

Then open your browser and navigate to `http://localhost:5173/html/` to view the demo page.

### Running the tests

Run the test suite:

```bash
npm test
```

### Running the tests with code coverage

Run the tests and generate a code coverage report:

```bash
npm run test:coverage
```

Coverage results will be written to the `./coverage` directory.
